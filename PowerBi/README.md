# 📊 Mobile Sales Dashboard - Power BI

## Project Overview
This project is an interactive Power BI dashboard developed to analyze mobile sales data and generate useful business insights. The dashboard provides information on sales performance, customer behavior, payment methods, ratings, and regional analysis.

## Objectives
- Analyze total sales performance
- Track quantity sold and transaction count
- Identify top-performing mobile brands
- Understand customer ratings and behavior
- Compare sales across cities and months
- Analyze payment method trends

## Tools & Technologies Used
- Power BI Desktop
- Excel Dataset
- Power Query
- DAX
- Data Visualization

## Dataset Information
The dataset contains the following fields:

- Transaction ID
- Date
- Day Name
- Brand
- Units Sold
- Sales Amount
- Customer Name
- Customer Age
- City
- Payment Method
- Customer Ratings
- Mobile Model

## Dashboard Features
### KPI Cards
- Total Sales: 769M
- Total Quantity: 19K
- Total Transactions: 4K
- Average Sales: 40K

### Visualizations Included
- Total Sales by City (Map)
- Total Sales by Mobile Model
- Average Customer Age by Mobile Model
- Customer Ratings Distribution
- Total Quantity by Month
- Total Sales by Day Name
- Payment Method Analysis
- Brand-wise Sales and Quantity Table

## Key Insights
- Apple, Samsung, Vivo, Xiaomi and OnePlus contribute significantly to sales.
- UPI, Credit Card, Debit Card and Cash are commonly used payment methods.
- Monthly sales trends help identify peak sales periods.
- Customer ratings provide customer satisfaction insights.
- City-level analysis shows regional sales patterns.
